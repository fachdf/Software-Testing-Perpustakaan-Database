A. Tools yang harus disiapkan
1. server : TOMCAT, atau yg lainnya bebas
2. Database mySQL
3. java, minimal jdk 1.8

B. cara jalanin aplikasi
1. buat database di dalam MYSQL dgn nama: perpustakaan
2. import/restore database yang sudah disiapkan ke dalam DBMS
3. copy war perpustakaanWebApp ke server pada folder 
   "....\tomcat\webapps"
4. jalankan server
5. ketikan URL Localhost:[PORT}//PerpustakaanWebApp/
